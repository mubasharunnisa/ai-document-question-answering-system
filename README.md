# ai-document-question-answering-system

An AI-powered web application that extracts text from PDF and image documents using OCR and answers user questions using transformer-based NLP models through an interactive Streamlit interface.

## Live Demo

🚀 Try out the live application here:

[Streamlit App](https://ai-document-question-answering-system-3hjrekjoue3vs7lf8mereg.streamlit.app/)

## Features

- PDF & Image document upload support
- OCR-based text extraction
- AI-powered Question Answering using Transformer models
- Interactive Streamlit web interface

## Installation

To run this project locally:

```bash
git clone https://github.com/mubasharunnisa/ai-document-question-answering-system.git

pip install -r requirements.txt
streamlit run streamlit_app.py
